# Changelog

All notable changes to Dub.io will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - Unreleased

### Added
- Initial release
- Video dubbing with ElevenLabs API
- YouTube video support via yt-dlp
- 29 language support for dubbing
- Dark mode CustomTkinter UI
- Transcript export functionality
- System tray integration (Windows)
- Session persistence
- Remote PIN activation system
