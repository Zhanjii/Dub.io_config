# Dub.io Configuration

Remote configuration repository for [Dub.io](https://github.com/Zhanjii/Dub.io).

## Files

- `app_access.json` - Application activation and remote configuration
- `CHANGELOG.md` - Auto-generated changelog from git commits

## Usage

This repository is automatically synced when new releases are created in the main Dub.io repository.

The changelog is generated from conventional commits and published here for in-app display.
